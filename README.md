## 📖 About This Specialization

The *Applied Data Science with Python* specialization introduces data science through the Python programming language. Across 5 courses, it covers statistical analysis, data visualization, machine learning, text mining, and social network analysis — all applied to real-world datasets using industry-standard libraries like pandas, matplotlib, scikit-learn, nltk, and networkx.

## 📚 Course Details

### 📌 Course 1 — Introduction to Data Science in Python
> Folder: [`Course1_IntroToDataScience`](./Course1_IntroToDataScience)

Fundamentals of data manipulation with pandas. Covers Series, DataFrames, merging, grouping, and cleaning messy real-world datasets.

---

### 📌 Course 2 — Applied Plotting, Charting & Data Representation
> Folder: [`Course2_DataVisualization`](./Course2_DataVisualization)

Principles of data visualization and applied charting with matplotlib. Covers design principles, scientific figures, and interactive plots.

---

### 📌 Course 3 — Applied Machine Learning in Python
> Folder: [`Course3_MachineLearning`](./Course3_MachineLearning)

Supervised and unsupervised learning with scikit-learn. Covers KNN, linear models, SVMs, decision trees, ensembles, and neural networks.

---

### 📌 Course 4 — Applied Text Mining in Python
> Folder: [`Course4_TextMining`](./Course4_TextMining)

Natural language processing fundamentals. Covers regex, NLTK, text classification, topic modeling, and semantic similarity.

---

### 📌 Course 5 — Applied Social Network Analysis in Python
> Folder: [`Course5_SocialNetworkAnalysis`](./Course5_SocialNetworkAnalysis)

Graph theory and network analysis with NetworkX. Covers connectivity, centrality, clustering, and link prediction.
